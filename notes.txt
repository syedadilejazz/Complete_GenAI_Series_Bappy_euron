What is LLMs?

These models are trained on massive amount of text data to learn patterns and entity relationship in the language.
It is a language model which is responsible for performing tasks such as:

-text to image generation
-image to text generation (Multimodal)

1. Closed Models
A closed model means:
You cannot download the actual model weights
You cannot see full training data
You cannot fully inspect training code
You mainly access it through an API or app

Example:
OpenAI GPT
Google Gemini
Anthropic Claude

Think of it like this
You can USE the AIBut you cannot OPEN the AI engine
Like:
You can drive a car
But cannot open factory design documents
What companies provide
Usually:
-Chat interface
-API access
-Some documentation
But NOT:
-Full model internals
-Weights
-Full training dataset
Advantages
-Very powerful
-Professionally maintained
-Usually best performance
-Safer and optimized
Disadvantages
-Expensive API cost
-No deep customization
-Internet required
-Company controls usage

2. Open-Weight Models
This is VERY important.
Open-weight does NOT always mean fully open-source.
It means:
The trained neural network weights are downloadable.
Example:
Meta Llama
Mistral AI Mistral
Technology Innovation Institute Falcon

What are “weights”?
Weights are the learned knowledge of the AI.
During training:
model learns patterns,numbers get adjusted,these learned numbers = weights

Simplified idea:
Training Data→Learning Process→Model Weights\text{Training Data} \rightarrow \text{Learning Process} \rightarrow \text{Model Weights}Training Data→Learning Process→Model Weights
Weights store:
language understanding,reasoning patterns,coding knowledge and relationships between words

Open-weight means you can:
-Download model
-Run locally
-Fine-tune it

Modify behavior
-Deploy on your own servers

But maybe NOT:
-Get training dataset
-Get full training code

Redistribute commercially freely
So:
Open-weight ≠ Fully open-source

The Transformer – a model that uses attention to boost the speed with which these models can be trained.

The encoder’s inputs first flow through a self-attention layer – a layer that helps the encoder look at other words in the input sentence as it encodes a specific word.

The embedding only happens in the bottom-most encoder. 
The abstraction that is common to all the encoders is that they receive a list of vectors each of the size 512 – 
In the bottom encoder that would be the word embeddings, but in other encoders, it would be the output of the encoder that’s directly below. 
The size of this list is hyperparameter we can set – basically it would be the length of the longest sentence in our training dataset.
